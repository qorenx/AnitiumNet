->Features
-Anime Watching
-Anime Calendar(There is a small problem.)
-Adding Anime Info and Episodes Automatically
-Adding Sections Manually
-Episode Download
-Add Season(Only, there are few problems.)
-Community
-Comment report system
-Add anime favorites etc.
